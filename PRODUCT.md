# Post for Me - Product Overview

## What is Post for Me?

Unified API platform for social media integrations: posting, scheduling, media processing, OAuth, feeds, and analytics across 9 platforms (TikTok, Facebook, Instagram, X, LinkedIn, Pinterest, Bluesky, Threads, YouTube) through a single API.

## Target Customers

1. **Social Media Schedulers** - Reduce integration overhead, ship faster with more platforms
2. **AI Content Tools** - Generate and post text/images/video across platforms seamlessly
3. **Marketing Teams** - Manage multiple clients, specialized workflows, custom reporting
4. **Mobile Games** - Share achievements/highlights to expand organic reach
5. **B2B SaaS** - Add social features without roadmap disruption

**User Personas**: Developers, Product Teams, Technical Founders, Engineering Managers needing fast integrations with minimal maintenance

## How It Works

Unified abstraction layer replacing 9 different APIs with unique auth flows, data formats, and quirks. Integrate once with Post for Me.

### Key Capabilities

- **Posting** - Create/schedule/publish text/images/video, single call to multiple platforms
- **Media Processing** - Auto-optimize for platform requirements, optional storage with cleanup
- **OAuth** - Simplified flows, auto token refresh, bring your own credentials or use ours
- **Feeds** - Fetch/display content, unified format across platforms
- **Analytics** - Track engagement, all platforms in one place, real-time + historical
- **Webhooks** - Real-time notifications for connections and post status

### Integration Methods

- **REST API** - Single entry point, consistent interface, docs at api.postforme.dev
- **SDKs** - TypeScript, Python, Go, Ruby, Kotlin with auto auth handling
- **MCP Server** - AI tool integrations, seamless content generation pipelines

### Developer Experience

One API replaces dozens. Example code gets you live same day. Handles platform quirks, consistent errors, access to unique features.
